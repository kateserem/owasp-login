# OWASP Login
This project is a simple login form developed using HTML, CSS< and JavaScript

## Features
- email and password input fields
- Client-side validations (email must include @ symbol and password must be 8 characters long at least)
- Basic input sanitization, preventing script injections

## Security
- Prevents common attacks such as SQL Injection and Cross-Site Scripting (XSS) by sanitizing special characters

## Technologies Used
- HTML
- CSS
- JavaScript
